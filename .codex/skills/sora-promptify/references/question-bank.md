# Question Bank (CLARIFY Mode)

Use this bank to select an *adaptive* question per CLARIFY round.
The intent is to be interactive without being prescriptive: ask what matters for
this scene, then move on.

Important: the entries below are a **pool**, not a script.
- In most rounds, you should **curate** a question into two strong creative directions (A/B) plus `Other: …`,
  instead of reading all options verbatim.
- Only present the full 4–5 option menu when the user asks for it or the nuance is critical.

Formatting requirements:
- Every question is multiple-choice.
- Include a default choice (mark with `[default]`) when sensible.
- Include a recommended choice when helpful:
  - Add `Recommended: <letter> — <one-line rationale>` immediately under the option list.
- Keep answers easy to type: A/B/C/D (use E only when needed).
- End with reply hint: `Answers: 1B 2A 3C ...`
- Prefer including an “Other: …” option so the user can escape the menu.

## Deterministic Selection Heuristic
Pick the single highest-priority unanswered question in this order:
1) Guardrails: must-haves / must-nots, safety + content boundaries, privacy/faces, text/logo + screen/UI depiction policy.
2) Missing anchors that would force story invention or mismatch intent: output format (if not default), who/where/what happens + primary subject of attention + ending beat + self-awareness + blocking/movement.
3) Filming perspective + how it’s filmed (camera holder, start/end framing, zoom/stability, allowed moves/stabilization).
4) Tone/genre mix + platform vibe (so the prompt matches intent).
5) If `dialogue_policy=preserve`: exact-vs-invented + content focus + timing + voice/performance + speaker binding + language/style constraints.
6) Shot plan + style + color grade + depth of field + intensity (unless already locked by user).
7) Cast complexity + background activity policy + intervention specifics (if chaos/crowd/intervention materially matter).
8) Environment type / time-of-day / weather + background detail density + wardrobe/prop anchors (if it changes lighting/audio/continuity).
9) Continuity strictness + audio focus/music policy + audio realism (refinements).
10) Physics mode (only if stunts/scale/implausibility appear).

If the user asks to bundle questions, pick up to 5 using the same priority list.
