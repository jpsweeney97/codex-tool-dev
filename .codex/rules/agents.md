# Rules: Agents

These rules are **blocking**. Read them before creating or editing any file under `.codex/agents/`.

## Required sections

Agent documents must include:

1. Scope
2. Non-goals
3. Tool usage rules
4. Output format
5. Escalation and approval rules

## Safety

- Do not perform destructive actions without explicit approval.
- Prefer smallest possible changes.
- If uncertain about scope or correctness, ask for clarification or run verification steps.
